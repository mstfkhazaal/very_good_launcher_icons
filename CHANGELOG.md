# Changelog

## 0.1.0

- Initial release.
- Add Android launcher icon generation for Very Good CLI flavored projects.
- Add support for `main`, `development`, and `staging` Android source sets.
- Add support for separate normal, round, and Play Store Android icons.
- Add iOS and macOS Very Good CLI `.icon` Logo.svg updates.
- Add iOS and macOS LaunchImage.imageset generation.
- Add Windows `.ico` generation.
- Add Web favicon and PWA icon generation.
- Add optional SVG-based icon composition using `resvg`.
